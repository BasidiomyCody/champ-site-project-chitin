What is CHAMP?

A community hub for coordinating mycology projects, events, and resources.

Core sections

  • News: longer updates and announcements
  • Calendar: upcoming events + event detail pages
  • Gallery: community images
  • Links: living list of resources
  • Feedback: suggestions + event proposals
  • Maps: future: foray spots, cemetery lichen routes, etc.

How the site works

  • The front page is built from the same JSON used by each section.
  • Events + Links are generated from simple .txt files in /content (then compiled into /data).
  • News is maintained in data/news/news.json (typically generated by a pipeline that ingests email or docs).
